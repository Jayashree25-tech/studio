# **App Name**: E-Reader Oasis

## Core Features:

- EBook Catalog Display: Display a catalog of eBooks with cover images, titles, authors, and prices.
- Shopping Cart: Shopping cart functionality, allowing users to add and remove eBooks.
- EBook Search and Filtering: Provide a way to filter or search eBooks by title, author or genre.
- Payment Processing: Integrate with a payment gateway to handle purchase transactions.
- AI Recommendation: Generate an book recommendation using the history of user searches as a tool.
- Login Page: Allow users to log in, providing fields for their name and phone number.
- Sign Out Page: Display a sign-out page with the message: THANK YOU FOR VISITING OUR E READER SHELF !! WAITING TO SEE YOU SOON
- Welcome Page: Display a welcome page with the message: HELLO WELCOME TO E READER BOOK SHELF

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calm and trustworthy ambiance, relating to knowledge and serenity.
- Background color: Very light blue (#F0F8FF) to provide a clean, non-distracting backdrop, remaining in the blue hue for consistency.
- Accent color: Light pink (#FDD7E4) for highlights and interactive elements, offering a gentle contrast while maintaining a connection to the primary color.
- Body and headline font: 'Literata', serif. Use this font for longer readability.
- Simple, clean icons for navigation and actions.
- Clean and responsive layout, optimized for different screen sizes.
- Subtle animations for transitions and user feedback.
