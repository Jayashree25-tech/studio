# E-Reader Oasis 📚✨

Welcome to **E-Reader Oasis**, your digital sanctuary for discovering and managing your next favorite books. Built with a clean, modern interface, this platform offers an intuitive browsing experience for both registered members and guest users alike.

## 🚀 Features

*   **Curated Book Catalog:** Explore an extensive, hand-picked collection of titles across various genres.
*   **Flexible Access Modes:** 
    *   **Secure Authentication:** Create an account or log in to personalize your reading shelf.
    *   **Guest Browsing:** Start exploring the library instantly without creating an account.
*   **Intuitive Shopping Cart:** Seamlessly manage and save books you're interested in purchasing or borrowing.
*   **Responsive & Clean UI:** Designed with a distraction-free layout optimized for reading and seamless navigation.

---

## 🛠️ Tech Stack

*   **Frontend:** React / Next.js (Tailwind CSS for styling)
*   **Icons:** Lucide React (or equivalent)
*   **Deployment:** Vercel
